PS 130 Code Club

This year at Code Club, an enthusiastic group of 4th graders learned the basics of coding. We used block coding languages as well as the languages most common on the web. We experimented with technologies such as machine learning and 3D design. We wrote code for tiny computers called micro:bits. Most importantly, we studied the impact that people who build technology can have on solving problems in the world.
